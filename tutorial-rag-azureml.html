Which Embeddings Model to use?#

There are currently two supported Embedding options:

  • OpenAI: text-embedding-ada-002, which can handle up to 8191 tokens. It can be accessed using Azure OpenAI or directly through OpenAI. If you don’t have an existing Azure OpenAI instance, AzureML provisions a default one for you called Default_AzureOpenAI. It provides high quality embeddings, one aspect to take into consideration, though, is the cost/quota available for the model.

  • HuggingFace offers a wide range of models. The MTEB Leaderboard ranks the performance of embeddings models on a few axis, though not all models can be run locally.
